# SRT - Workflow reel court V1

1. Remplir `WIFI_SSID`, `WIFI_PASSWORD`, `TELEGRAM_BOT_TOKEN`, `TELEGRAM_CHAT_ID`.
2. Flasher `borne_srt_entree_workflow_reel_court_v1.ino` sur la borne entree.
3. Flasher `borne_srt_sortie_workflow_reel_court_v1.ino` sur la borne sortie.
4. Activer le hotspot telephone en Wi-Fi 2.4 GHz.
5. Verifier Telegram : message `BORNE SRT EN LIGNE`.
6. Creer un vrai dossier dans le Terminal SRT.
7. Scanner un vrai QR CSR Tag.
8. Passer devant la borne entree.
9. Verifier Telegram : retour detecte.
10. S'eloigner 8 secondes.
11. Passer devant la borne sortie.
12. Verifier Telegram : sortie detectee.
13. Bouger / secouer le CSR Tag proche d'une borne.
14. Verifier Telegram : mouvement detecte.
15. Ouvrir Terminal SRT et retrouver CSR Tag -> plaque -> dossier.
